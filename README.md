## ✨ الميزات الرئيسية

### 📚 الحفظ والمراجعة
- 🧠 **التكرار المتباعد (SM-2)**: خوارزمية علمية لتثبيت المحفوظات في الذاكرة طويلة المدى
- 📋 **تصنيفات متعددة**: القرآن، الحديث، المفردات، الدروس، الشعر
- 🔁 **جلسات المراجعة**: إشعارات ذكية بالمحفوظات التي تحتاج مراجعة
- ⭐ **المفضلة**: حفظ المحفوظات المهمة بسهولة

### 🕌 الأدوات الإسلامية
- 🌅 **حصن المسلم**: أذكار الصباح والمساء والنوم وبعد الصلاة
- ⭐ **الأسماء الحسنى**: 99 اسماً مع معانيها الكاملة والبحث فيها
- 📿 **السبحة الإلكترونية**: عدّاد للتسبيح بـ 6 أذكار مختلفة
- ⏲️ **مؤقت الحفظ**: جلسات Pomodoro للتركيز والحفظ
- 🎯 **مسابقة الحفظ**: اختبار ذاتي لقياس قوة الحفظ
- 💡 **نصائح الحفظ**: 15 نصيحة ذهبية للحفظ المتقن
- 🏆 **الإنجازات**: 14 شارة قابلة للفتح
- 🎯 **التحديات**: أهداف يومية وأسبوعية
- ✨ **تذكرة اليوم**: آية وحديث ودعاء يومي

### 📖 المصحف الشريف
- 114 سورة كاملة مع بياناتها
- 30 جزءاً مفصلاً
- بحث ذكي عن السور
- واجهة إسلامية فاخرة

### 🎨 التخصيص
- 🌓 **الوضع الليلي الكامل**
- 🎨 **6 ألوان رئيسية فاخرة** (فيروزي، ذهبي، زمردي، بنفسجي، عنابي، أزرق)
- 📝 **تخصيص حجم الخط** (80% - 160%)
- 🎯 **الأهداف اليومية** القابلة للتعديل
- 🔔 **تفعيل/إيقاف التذكيرات**

### 💾 النسخ الاحتياطي
- 📤 **تصدير البيانات** إلى JSON
- 📥 **استيراد البيانات** من نسخة سابقة
- 💫 حفظ السلاسل والإحصائيات

### 📊 الإحصائيات
- 📈 مخططات بيانية تفاعلية
- 🔥 عدّاد الأيام المتتالية (Streak)
- ⏱️ إجمالي دقائق الدراسة
- 🎓 عدد المحفوظات المُتقنة
- 📅 إحصائيات يومية وأسبوعية

## 🏗️ البنية التقنية

```
lib/
├── data/              # البيانات الثابتة (أذكار، أدعية، أسماء حسنى)
├── models/            # نماذج البيانات (Hive)
├── providers/         # إدارة الحالة (Provider)
├── screens/           # شاشات التطبيق
│   ├── azkar/         # شاشات الأذكار
│   ├── quran/         # شاشات المصحف
│   ├── tools/         # الأدوات الإسلامية
│   ├── settings/      # الإعدادات
│   └── hifz/          # جلسات الحفظ
├── services/          # الخدمات (تخزين، تكرار، نسخ احتياطي)
├── theme/             # النمط والثيم
└── utils/             # أدوات مساعدة
```

### 🔧 التقنيات المستخدمة
- **Flutter 3.35.4** - إطار العمل
- **Dart 3.9.2** - لغة البرمجة
- **Provider 6.1.5** - إدارة الحالة
- **Hive 2.2.3** - التخزين المحلي (NoSQL)
- **fl_chart 0.69.2** - المخططات البيانية
- **percent_indicator 4.2.3** - مؤشرات النسبة
